ISCE is a Synthetic Aperture Radar (SAR) processing library on which data processing workflows for the NISAR mission is built. ISCE is released under the XXX license. As a library, it provides a variety of data structures (C++/ Python) that are used frequently in SAR/ InSAR data processing. ISCE also comes with a variety of useful data processing pipelines for SAR data processing.
